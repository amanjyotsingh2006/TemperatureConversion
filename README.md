# Temperature Converter
#### A small, beginner-friendly web app that converts temperatures between Celsius and Fahrenheit. Built with plain HTML, CSS and JavaScript — great for learning DOM manipulation and simple form handling.

## Open index.html in a browser (double-click the file or serve it from a static server) and:
### 1.Type a number in the input field.
### 2.Choose Celsius to Fahrenheit or Fahrenheit to Celsius.
### 3.Click Convert — the result appears in the disabled result field.
